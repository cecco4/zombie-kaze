#define DEBUG_CONSOLE

Questo file contiene la definizione della Debug console.

E' una speciale interfaccia attivabile solo in modalità di debug che consente di ricavare informazioni in tempo reale su player, mappa e nemici. è possibile inoltre scegliere quale livello giocare.

Per attivare questa modilità, è necessario compilare il progetto con "make debug" e una volta in gioco premere "p".
